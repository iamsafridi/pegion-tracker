# Quick Start: Ring Number Search

## How to Use (3 Simple Steps)

### Step 1: Click the Button
Look for the **pink/purple "Search Ring History"** button in the controls section (next to the search box).

### Step 2: Enter Ring Number
Type the ring number you want to search for:
- Example: `24-52228-h`
- Works with partial numbers too: `24-52`

### Step 3: View Results
See complete pigeon history including:
- Total races participated
- Best position achieved
- Average performance
- Complete race-by-race breakdown
